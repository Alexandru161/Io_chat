# Io_chat 💬

Io_chat is a real-time chat application designed for fast and efficient communication between users. The project demonstrates a full-stack architecture using modern technologies, including backend API, database integration, and containerized deployment.

---

## 🚀 Features

- Real-time messaging
- User-based communication
- REST API for managing data
- Scalable backend architecture
- Containerized environment using Docker
- Ready for cloud deployment

---

## 🛠 Tech Stack

**Backend**
- NestJS
- Node.js

**Database**
- PostgreSQL
- Sequelize ORM

**DevOps**
- Docker & Docker Compose
- NGINX (reverse proxy)

**Other**
- REST API
- Environment-based configuration
